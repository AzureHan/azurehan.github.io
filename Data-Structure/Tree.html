    # 树

    # Binary Tree ( 二叉树 )

    • 所有节点最多拥有两个子节点

    • 左子树的键值小于根的键值，右子树的键值大于根的键值

    # AVL Tree ( 平衡二叉树 )

    基于 AVL 算法。

    • 非叶子节点最多拥有两个子节点

    • 非叶子节点的值，大于左边子节点、小于右边子节点

    • 树的左右两边的层级数相差不会大于一

    • 没有值相等重复的节点

    # B-Tree ( 平衡多路查找树 )

    • 非叶子节点拥有多个子节点，所有叶子节点均在同一层
    • 叶子节点包含了 关键字、关键字数据记录的指针和 指向其子节点的指针
    • 非叶子节点的关键字数量是其叶子数量少一
    • 所有节点关键字是按递增次序排列，并遵循左小右大原则

    # B+Tree ( 平衡多路查找树 - 升级版 )

    • 非叶子节点只进行关键字索引，不会存实际的关键字数据记录的指针
    • 所有数据记录的指针必须要到叶子节点才能获取到，所以每次数据查询的次数都一样
    • 节点关键字数量和其子节点个数相等
