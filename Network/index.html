# Web 优化

# 优化页面加载速度的方法

  1. 前端减少引入 js \ css 文件的数量
  2. 前端压缩 js 文件
  3. 后端使用缓存避免重复的数据库查询
  4. 服务器使用 CDN 加速
  5. 服务器开启 Gzip 压缩
  6. 使用浏览器缓存

# 访问很慢的情况

  1. 客户端与服务器之间有网络问题
  2. 服务器有大量并发请求导致资源不足
  3. 程序有向外请求
  4. 程序处理大量事务
  5. 大量缓存没有命中
  6. 数据库有慢查询和阻塞等待
  7. 分布式系统故障
