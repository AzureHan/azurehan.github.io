    # Socket

    进程之间在网络中可以通过 Socket 来通信。

    # Socket 通讯流程

    服务端首先建立 Socket ，然后绑定端口、监听端口，客户端首先也要建立 Socket ，然后请求连接服务端。服务端接受后开始接收客户端发送的数据，最后客户端关闭连接，服务器也随之关闭。

    # Socket 的基本操作

    • socket 函数
    • bind 函数
    • listen 函数 \ connect 函数
    • accept 函数
    • read 函数 \ write 函数 \ ...
    • close 函数
